What is Velo

AI-native video messaging, end to end

Velo is an AI-powered tool that transforms raw screen recordings, uploads, PDFs, slides, and URLs into polished, share-ready video messages — no manual editing required.

Learning Path

Your Velo learning journey

An opinionated, experience-optimized path to go from zero to Velo expert. Designed for practitioners who learn by doing.

1
Read the Introduction & Quickstart Docs

Start at docs.usevelo.ai. Read the Introduction and Quickstart pages. Understand the four input modes and what makes Velo different from Loom or Camtasia. Orient yourself to the mental model before touching the app.

schedule20–30 min
2
Create your first Velo via Screen Capture

Install the Chrome Extension. Record a simple 2–3 minute product walkthrough of anything familiar. Accept the AI-generated script without editing it the first time — just see the full output pipeline end to end. Share via link.

schedule45–60 min
3
Explore the Editor — Every Panel

Open the output from step 2 in the editor. Systematically click through every right sidebar panel and every timeline track. Change the background, adjust cursor style, add a callout, try a zoom block. You're mapping the terrain, not producing anything.

schedule30–45 min
4
Try all four Input Modes

Create one Velo from each mode: upload an existing MP4 (a Zoom recording works well), upload a PDF deck, and paste a URL for a simple SaaS product. This gives you intuition for when to use each mode vs. the others.

schedule1.5–2 hrs
5
Create & Calibrate Your VeloTwin

Go to the VeloTwin section and create your custom avatar. Read the best practices first (lighting, background, eye contact, voice pacing). Create it, then produce one Velo using your VeloTwin face and voice. Evaluate the output critically and tweak if needed.

schedule45–90 min
6
Master Script Editing & Context Prompting

Produce a real-world demo for your product or project. This time: edit the AI script before generating, use the rewrite-with-context feature to align tone (e.g., "executive audience, skip technical detail"), and try at least 2 retries to understand how different scripts diverge.

schedule1–2 hrs
7
Build a Production-Quality Demo

Create a demo you'd actually send to a prospect or stakeholder. Use brand kit, a proper intro card, custom cursor, appropriate zoom blocks and spotlights, clean masking of any sensitive data, and your VeloTwin. Share it and collect feedback.

schedule2–3 hrs
8
Develop Workflow Patterns & Templates

After 5–10 real demos, identify your repeatable patterns: which input mode works best for your content type, what brand kit settings you prefer, standard zoom/spotlight sequences, script tone guidance you always provide. Document these as your personal Velo playbook.

scheduleOngoing

Proficiency Estimates

How long to become a Velo expert?

For an experienced developer who already understands AI coding/tooling best practices — context management, prompting, workflow structuring. Hours are cumulative active practice time, not calendar time.

5
/ 10 — Proficient
3–5 hrs
cumulative practice
  • Completed all 4 input modes at least once
  • Can navigate every editor panel
  • Understands VeloTwin setup flow
  • Produces shareable output reliably
  • Knows when to use each input mode
  • Can edit & retry scripts confidently
8
/ 10 — Advanced
12–20 hrs
cumulative practice
  • Consistent production-quality output
  • Custom VeloTwin calibrated & polished
  • Brand kit applied systematically
  • Strategic use of zoom, masks, spotlight
  • Context prompting produces predictable tone
  • Personal workflow playbook documented
  • Agentic link-to-video used confidently
10
/ 10 — Expert
40–60 hrs
cumulative practice
  • Can teach others the full Velo system
  • Builds reusable demo templates & personas
  • Pushes edge cases (long-form, complex flows)
  • Understands all plan limits & workarounds
  • Masters every annotation tool precisely
  • Produces broadcast-ready content first take
  • Integrates Velo into broader content pipeline
  • Handles troubleshooting independently
Why these estimates? Velo's core loop is shallow and fast — the 5/10 threshold is genuinely low because the AI handles the hard parts. Getting from 5 → 8 requires real repetition to develop taste for script quality and output calibration. The 8 → 10 jump is mostly about accumulated edge case experience and building systematic reuse patterns, not learning new buttons.
Pricing

Plans & Velo Minutes

All plans include full editor access and watermark-free sharing links. Velo Minutes are consumed by the length of recordings and videos generated each month.

Free
$0/month
  • 15 Velo minutes / month
  • Full editor access
  • Share links — no watermark
  • 1 avatar video/month
  • Public VeloTwins
  • Downloads watermarked (remove for $9.99)
Pro
$20/month
  • 60 Velo minutes / month
  • Up to 3 min avatar / video
  • All public avatars
  • No watermarks on downloads
  • 3 script retries / video
  • 3 voice regenerations / video
  • 5 link-to-video / month
Ultra
$200/month
  • 600 Velo minutes / month
  • Up to 30 min avatar / video
  • Custom VeloTwins (your face & voice)
  • 10 script retries / video
  • 10 voice regenerations / video
  • Unlimited link-to-video
Which plan should you start on? Free is enough to evaluate Velo and complete the learning path through step 4. You'll need Pro to use the agentic link-to-video mode and get meaningful script retry budget. Ultra is only warranted at high-volume production (daily demos, long avatar segments, or team-wide usage).
